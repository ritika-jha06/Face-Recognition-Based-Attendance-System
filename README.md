# How the System Works

1.Register Faces

    User runs add_faces.py

    Camera captures face images
    
    Face data is stored in .pkl files
    
2.Recognize & Mark Attendance

    User runs app.py
    
    Live camera feed opens inside the UI
    
    Faces are detected and recognized
    
    Attendance is automatically saved with date & time
